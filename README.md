# Energy Usage Tracker AI

An advanced AI-powered Energy Usage Tracker that monitors electricity consumption in real time, predicts future energy usage, provides smart AI recommendations, and helps users reduce power consumption efficiently.

---

## ⚡ Real-Time Energy Monitoring

* Live power usage tracking
* Daily and monthly consumption analytics
* Real-time electricity bill estimation
* Dynamic device monitoring

* Predicts future energy consumption
* Detects high power usage patterns

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/energy-usage-tracker.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd energy-usage-tracker
```

## 3️⃣ Install Dependencies

```bash
npm install
```

## 4️⃣ Configure Environment Variables

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection
```

## 5️⃣ Start Development Server

```bash
npm start
```
